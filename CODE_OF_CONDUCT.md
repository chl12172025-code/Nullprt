# Nullprt Code of Conduct

## Reporting Channels
- Primary: security@nullprt.org
- Backup: conduct@nullprt.org
- For urgent incidents: `#conduct-emergency` moderator channel.

## Handling Process
1. Acknowledge report within 24h.
2. Assign two maintainers (or external neutral reviewers if conflict exists).
3. Collect evidence and statement from involved parties.
4. Issue decision within 7 days when possible.

## Enforcement Actions
- Warning
- Temporary moderation or contribution restrictions
- Temporary ban
- Permanent ban

## Confidentiality Commitment
- Reporter identity is kept confidential unless explicit consent is given.
- Incident records are access-controlled and retained for audit only.
- Public incident reports are anonymized.
