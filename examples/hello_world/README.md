# hello_world

用途：演示多文件项目组织、单元测试、基准测试。

## 项目结构
- `src/main.nprt`: 程序入口
- `src/math.nprt`: 多文件模块示例
- `tests/unit_test.nprt`: 单元测试示例
- `bench/bench_fib.nprt`: 基准测试示例

## 运行
- 编译：`./build.sh` 或 `.\build.ps1`
- 预期返回码：`0`

## 性能基准数据（示例）
| case | iterations | avg_us | p95_us |
|---|---:|---:|---:|
| fib(20) | 1000 | 310 | 342 |
| add loop | 100000 | 48 | 55 |

## 安全注意事项
- 仅在受信环境运行示例二进制。
- 不要把示例中的调试开关用于生产发布配置。

## FAQ
- Q: 为什么要拆分 `main` 和 `math`？
  - A: 展示模块边界和可测试单元。
- Q: 单元测试如何接入 CI？
  - A: 在 `tests/` 下维护可执行测试入口并加入脚本。
