# 许可证验证演示

演示硬件绑定授权流程：离线激活、心跳保活、吊销检测。

## 本示例新增点
- 离线激活示例（请求码/激活码）
- 心跳保活示例（定时上报）
- 许可证吊销示例（吊销列表版本）

## 性能基准数据（示例）
| case | avg_ms | p95_ms |
|---|---:|---:|
| offline activate verify | 3.2 | 4.8 |
| heartbeat roundtrip | 18.5 | 27.1 |
| revocation check | 2.1 | 3.0 |

## 安全注意事项
- 激活码与设备指纹必须加密存储。
- 心跳接口必须启用 TLS 与重放防护。

## FAQ
- Q: 离线激活为什么仍需要周期联网？
  - A: 用于同步吊销状态与策略更新。
