# Nullprt Technical Blog Program

## Publishing Platform Accounts
- Register official accounts on:
  - GitHub Blog/Pages
  - Dev.to organization
  - Medium publication
  - Zhihu column (CN community)

## Content Calendar
- Weekly short update: toolchain progress and fixes.
- Bi-weekly deep dive: compiler/LSP/debug/profiler internals.
- Monthly recap: roadmap outcomes and next votes.

## SEO Optimization
- Use stable slugs and canonical URLs.
- Add schema.org `Article` metadata.
- Maintain keyword clusters: `compiler`, `language server`, `debugger`, `profiling`, `Nullprt`.

## Comment and Interaction Management
- Triage comments every 48h.
- Label feedback as `question`, `bug-report`, `feature-request`, `roadmap-input`.
- Escalate technical questions into GitHub discussions when code changes are needed.
