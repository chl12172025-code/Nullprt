# v0.2.0-beta 发布说明

相较于 alpha：

- 发布对比链接：[v0.1.0-alpha...v0.2.0-beta](https://github.com/chl12172025-code/Nullprt/compare/v0.1.0-alpha...v0.2.0-beta)

- 增加开发者研究门控（编译期开关 + 运行时令牌 + 审计日志）
- 增加硬件标识读取、设备指纹与许可证绑定校验 API
- 增加 VM 环境信号检测 API 与开发者研究模拟覆盖能力
- 增加编译器保护 pass 框架（控制流变换、opaque predicate、字符串混淆、完整性标记、反调试标记）
- 增强 `nprt-pkg` 依赖解析（SemVer 约束、冲突信息、扩展 lockfile 字段）
- 增加 API 文档生成脚本、教程集合、示例项目集合与 GitHub Pages 工作流

## 安全公告（重点）

> SECURITY ADVISORY:
> 高风险能力仅允许开发者研究构建，并受三重门控保护。
> 发行版默认关闭相关能力，禁止用于攻击第三方系统或绕过授权验证。

已知限制：

- 部分跨进程内存操作在 Unix 平台当前返回 `unsupported`，用于避免越权实现
- Native backend 仍处于最小可验证阶段，后续继续扩展
